### Setting the project up:

##### install:
- node 6.4.0+
- npm 3.10.7+

##### commands:
```
git clone the SpatialMap repo
npm install --save
npm run start
```

in case errors appear you might need to add

```
npm install --save firebase
npm install --save react-dimensions
npm install --save react-parallel-coordinates
```
